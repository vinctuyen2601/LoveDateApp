# Icons Download List

Tải từ **[icons8.com](https://icons8.com)** — format **PNG**, size **80px**.

---

## Style khuyến nghị

| Bộ | Style Icons8 | Lý do |
|---|---|---|
| Tags (birthday, family...) | **Color** hoặc **Fluent** | Full-color đẹp hơn monochrome nhiều |
| Special dates (Tết, Valentine...) | **Color** hoặc **Fluent** | Mỗi ngày có visual đặc trưng |
| Activities (restaurant, travel...) | **Color** hoặc **Fluent** | Dùng cho màn hình gợi ý kế hoạch hẹn hò |
| MBTI (4 nhóm tính cách) | **Color** hoặc **Fluent** | Icon đại diện cho từng nhóm tính cách |

> **Lưu ý UI:** Tag color (`#FF6B6B`, `#FF69B4`...) vẫn dùng cho **background bubble** quanh icon.
> Icon giữ nguyên màu sắc gốc từ Icons8 — không dùng `tintColor`.

```
┌───────────────────────┐
│  bg: tag.color + 15%  │  ← màu tag dùng ở đây
│   🎂  full-color PNG   │  ← icon giữ màu đẹp gốc
└───────────────────────┘
```

---

## Hướng dẫn tải

1. Vào [icons8.com](https://icons8.com) → search từ khoá bên dưới
2. Filter Style: **Color** (hoặc **Fluent** nếu thích outline hơn)
3. Click icon → **Download PNG** → chọn size **80px**
4. Đổi tên đúng như cột "File name"
5. Đặt vào thư mục tương ứng
